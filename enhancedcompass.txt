=====================================
ENHANCED COMPASS GUIDE 
=====================================
A player's guide to the /enhancedcompass command

By SupaFloof Games
=====================================
PAGE 1
=====================================
WHAT IS THIS?

EnhancedCompass lets you point your compass to any structure or biome in Minecraft.

When you hold a compass, a boss bar appears showing the target name and distance.
=====================================
PAGE 2
=====================================
BASIC USAGE

1. Get a compass
2. Run a search command
3. Hold the compass
4. A boss bar appears at the top of your screen
5. Follow the compass!
=====================================
PAGE 3
=====================================
MAIN COMMANDS

/enhancedcompass help
Shows command help

/enhancedcompass <structure>
Points compass to that structure

/enhancedcompass biome <biome>
Points compass to that biome

/enhancedcompass current
Shows your current target
=====================================
PAGE 4
=====================================
SPECIAL SEARCHES

/enhancedcompass village
Finds the nearest village of ANY type (plains, desert, snowy, etc.)

/enhancedcompass anything
Finds the nearest structure of ANY type

BIOME COMMAND

Example:
/enhancedcompass biome dark_forest

Result:
"Compass now pointing to Dark Forest biome!"
"Distance: 523 blocks"
=====================================
PAGE 10
=====================================
BOSS BAR COLORS

Aqua target name + Yellow distance:
Target is in your dimension

Red target name + Red warning:
Target is in a different dimension
=====================================
PAGE 11
=====================================
BOSS BAR BEHAVIOR

- Appears when you hold a compass (main or off hand)
- Updates every half second
- Disappears when you put the compass away
- Shows real-time distance in blocks

"Distance: 456 blocks"
=====================================
PAGE 17
=====================================
TAB COMPLETION

Start typing a command and press TAB.

Example:
/enhancedcompass anc[TAB]

Completes to:
/enhancedcompass ancient_city

Works for structures AND biomes!
=====================================
PAGE 18
=====================================
BIOME TAB COMPLETION

Type:
/enhancedcompass biome [TAB]

Shows all available biomes:
dark_forest
cherry_grove
mushroom_fields
...and more!
=====================================
PAGE 19
=====================================
PERSISTENT TARGETS

Your compass target is saved automatically.

It stays set even if you:
- Log out
- Die
- Server restarts

To change target, just search for a new structure or biome.
=====================================
PAGE 20
=====================================
DISTANCE EXPLAINED

Distance is "as the crow flies" - a straight line from you to the target.

Includes height (Y-level).

Actual walking distance may be longer due to terrain.
=====================================
PAGE 21
=====================================
CROSS-DIMENSION

If you set a target in the Overworld, then go to the Nether:

Boss bar shows:
"End City - Not in same dimension"

Return to the correct dimension to see the real distance.
=====================================
PAGE 22
=====================================
STRUCTURE SEARCH

When you run a search command, the plugin:

1. Searches the world
2. Finds the nearest match
3. Points your compass to it
4. Shows you the distance
=====================================
PAGE 23
=====================================
PERMISSION REQUIRED

You need the permission:
enhancedcompass.use

If you see "You don't have permission" ask your server admin.
